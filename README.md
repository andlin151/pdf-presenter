# PDF to Impress.js

Un proyecto **quick & dirty**, simple y práctico que permite hacer una presentación con transiciones a partir de un PDF. Muy útil si sueles hacer tus presentaciones con Beamer o simplemente detestas PowerPoint.

## Características

- **Drag & Drop** - Simplemente arrastra tu PDF y listo
- **8 tipos de transiciones** - Horizontal, vertical, diagonal, espiral, zoom, rotación, cubo 3D y aleatorio
- **Colores de fondo personalizables** - Presets incluidos o elige tu propio color
- **Pantalla completa** - Presiona `Ctrl+F` para entrar/salir
- **Sin servidor** - Funciona completamente en el navegador
- **Sin dependencias locales** - Todo se carga desde CDN

## Uso

Clonar y colocar en su servidor web de preferencia. No hay más detalles al respecto.
O simplemente, entrar aquí.

## Controles de navegación

| Tecla | Acción |
|-------|--------|
| `→` `↓` `Space` | Siguiente slide |
| `←` `↑` | Slide anterior |
| `Ctrl+F` | Pantalla completa |
| `P` | Consola del presentador |

## Tecnologías

- [impress.js](https://impress.js.org/) - Motor de presentaciones con CSS3
- [PDF.js](https://mozilla.github.io/pdf.js/) - Renderizado de PDF de Mozilla

## Limitaciones

- PDFs muy grandes pueden tardar en procesar
- Requiere un navegador moderno con soporte para CSS3 transforms
